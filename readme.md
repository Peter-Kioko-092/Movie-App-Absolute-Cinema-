#  Absolute Cinema

**Absolute Cinema** is a sleek, responsive, and modern movie discovery web application. It leverages the **TMDB (The Movie Database) API** to provide real-time trending data, high-quality hero features with trailer integration, and a persistent personalized watchlist.

---

##  Features

*   **Cinematic Hero Section:** Dynamic top-trending movie feature with high-resolution backdrops, metadata, and instant trailer playback.
*   **Real-time Movie Search:** Instant search results with debounce logic to ensure a smooth user experience.
*   **Smart Filtering:** Filter movies by Genre, Release Year, or Sort by Popularity and Rating.
*   **Persistent Watchlist:** Add your favorite movies to "My List." Data is saved locally in your browser, so it’s there when you come back.
*   **Horizontal Trending Carousel:** A Netflix-style scrollable carousel featuring the top 10 movies of the week.
*   **Responsive Design:** Optimized for Desktop, Tablet, and Mobile with a glassmorphism UI.
*   **Absolute Aesthetic:** Dark mode by default with signature red neon glows and interactive button animations.

---

##  Tech Stack

*   **HTML5:** Semantic structure for optimal accessibility.
*   **CSS3:** Custom properties (variables), Flexbox, CSS Grid, and high-end animations/shadows.
*   **JavaScript (ES6+):** Object-Oriented approach using Classes to manage API state and UI logic.
*   **TMDB API:** The source of all movie data, posters, and trailers.
